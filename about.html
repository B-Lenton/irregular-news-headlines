About This Project

A JavaScript-focused random news generator created independently as part of the Just IT Skills Bootcamp, 2021. Read on to find out more.

Calling The Guardian's REST API

I researched a number of APIs before choosing the Guardian's, partly for its ease of use. The data is collected using fetch() inside a while loop, yielding a total of 500 news headlines.

View API

Compromise NLP

Compromise felt like the obvious choice for a fully frontend Natural-Language Processor. It is lightweight, simple to use, yet powerful. I parsed nine categories of words and refactored the results in varying ways; these results make up the entirety of every random headline.
